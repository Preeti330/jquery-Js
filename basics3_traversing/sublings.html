Sublings :
  • sublings() : finds all sublings
  • sublings('p') : finds all sublings of p
  • next() : returns next subling element
  • nextAll() : returns next all subling elements
  • nextUntill('tag') : returns next all subling elements till tag
  • prev() : returns prev subling element
  • prevAll() : returns prev all subling elements
  • prevUntill('tag') : returns prev all subling elements till tag
