Filters :
  • first() , last(),eq() : finds the elements on base of its postions
  • filter() , not : finds the elements on base of given creteria
  • first() : returns its first element
  • last() : returns last elements of tag
  • eq(index) : returns element on base of index
  • filter('.class') : filter the elements which meets the required cretira Ex: returns all elements with class
  • not('.class') : filter the elements which not meets the required cretira Ex: returns all elements not with class
