Jquery Dimentions

  • width() : which gives the width of an element (or sets the width of an elements)
  • innerWidth()/ outerHeight() : element width + padding (or sets the width of an elements)
  • outerWidth() outerHeight(): element width + padding + border (or sets the width of an elements)
  • outerWidth(true)/outerHeight(true) : element width + padding + margin + border (or sets the width of an elements)
