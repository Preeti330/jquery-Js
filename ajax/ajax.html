ajax - uses for sending the ajax request with ajax method and typiclly we use when request made with spcifications

Possible key values

  • async : a boolean value indicating where request happens in asycronusoully or not , default is true
  • type : indicates which request (get or post)
  • data : request data
  • dataType : response data type
  • url : request url
  • sucess(result,status,xhr) : function runs after sucessfull excution
  • error(error,status,xhr) : runs when errors occurs
  • complete(xhr,status) : runs after sucess and error
