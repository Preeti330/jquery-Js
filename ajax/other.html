• $.ajaxSetup(options) : sets the default values for ajax request,as in url , datatype,headers etc it can be mpdifed when we over ridden the method other voice it will be same for all request 
    •  $.ajaxSetup({
                    headers: {
                        'Authorization': 'Bearer your_token_here',
                        'Content-Type': 'application/json'
                    }
                });
    •  $.ajaxSetup({
                        timeout: 5000 
                 });
    •  $.ajaxSetup({
                        dataType : json
                 });
    •  $.ajaxSetup({
                        error : function(error){
                            console.log("Error "+ error)
                        }
                 });
  • $.ajaxError() : runs when sme error happens 
    •                         
                            $.ajaxError(function(error){
                                console.log("Error "+error);
                            })
    • $.ajaxCompletes() : excutes when ajax completes its excetuions
